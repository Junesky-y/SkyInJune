# Lost Volyn Kouture — Онлайн-магазин одягу
> Простий веб-магазин на HTML/CSS/JS з каталогом, кошиком, фільтрами та детальним переглядом товарів.

[Перейти до демо ↗️](https://junesky-y.github.io/SkyInJune/)

[![Deployment Status] (https://github.com/Junesky-y/SkyInJune/deployments)

---

##  Про проект
LVK — веб-додаток для користувачів, який дозволяє:

- Переглядати каталог одягу з категоріями та стилями
- Використовувати фільтри за розміром, кольором та ціною
- Переглядати детальні фото та опис товарів
- Додавати товари до кошика та оформлювати замовлення
- Отримувати рекомендації схожих товарів

Проєкт реалізовано на чистому HTML/CSS/JS без фреймворків.

---

##  Як встановити проект

git clone https://github.com/Junesky-y/SkyInJune.git
cd SkyInJune
npm install

## Приклад використання


1. Переглянь каталог та використай фільтри за категорією, кольором, розміром або ціною.

2. Клік на товар відкриває детальну сторінку з фото та описом.

3. Додай товар у кошик і перевір суму замовлення.

4. Оформи замовлення через тестову форму оплати.

5. Переглянь рекомендовані товари на основі обраних.

## Development setup
Щоб підготувати середовище для розробки та перевірити функціонал:

1. Встановіть всі залежності:

npm install

2. Запустіть юніт-тести для перевірки роботи функцій:

npm test

Проєкт використовує Jest для тестування функції changeLargeImage() у середовищі jsdom.
